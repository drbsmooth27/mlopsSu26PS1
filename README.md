# Movie Review Sentiment Analyzer

## How to run locally

- Clone the repository: `git clone https://github.com/drbsmooth27/mlopsSu26PS1`
- Navigate into the project folder: `cd mlopsSu26PS1>`
- Install the required packages: `pip install -r requirements.txt`
- Make sure `sentiment_model.pkl` is in the same folder as `app.py`
- Run the app: `streamlit run app.py`
- Your browser should open automatically at `http://localhost:8501`
